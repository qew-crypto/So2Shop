# 🤖 STAND FX Telegram Bot

Telegram-бот продажи приватных читов Standoff 2. Полностью демо-режим.

## 🎯 Возможности

### 👤 Пользователю:
- 💎 Каталог из 3 тарифов (BASE 199₽, VIP 499₽, PREMIUM 1299₽)
- 💳 Пополнение баланса через СБП / Карту / Крипту + ежедневный бонус +100₽
- 🎫 Генерация и активация ключей
- 📥 Скачивание .apk (демо)
- 📜 История операций
- 🌟 Геймификация (500₽ стартовый бонус)

### 🎛 Админ-панель (`/admin`):
- 📊 Полная статистика сервиса
- 👥 Список всех юзеров с пагинацией
- 🎫 Список всех ключей + генерация новых
- 💳 Журнал всех транзакций
- 💰 Ручное начисление/списание баланса
- 🎁 Выдача тарифа в один клик (с авто-уведомлением юзеру)
- 🚫 Бан/разбан + ⚠️ предупреждения
- 📢 Рассылки по сегментам (все / активные / VIP+)

## 🚀 Запуск

```bash
pip install -r requirements.txt
# В config.json вставь BOT_TOKEN и ADMIN_IDS
python main.py
```

## 🔐 Лимиты

- 🔒 Нельзя купить тариф ниже активного
- 🔑 Ключ привязывается к устройству (один ключ = одна активация)
- 🚫 Баннинг за нарушения
- 🎁 Бонус +100₽ раз в сутки

## ⚙️ Настройка

В `config.json`:
- `BOT_TOKEN` — токен от @BotFather
- `ADMIN_IDS` — массив Telegram ID админов
- `START_BALANCE` — стартовый бонус новым юзерам
- `DEMO_MODE` — флаг демо-режима (оплаты заглушены)

## 💡 DEMO-режим

Все платежи имитируются. Реальные деньги не списываются. Кнопки оплаты возвращают успех с задержкой 0 сек. В рабочей версии нужно подключить реальные API (CryptoBot, YooKassa и т.п.).
